# Task-runner

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Description

Simple react application, where user can add, remove tasks, and give them a priority range. 

## Installation

1. Download zip or clone repo
2. Navigate to folder with application in your terminal
3. Write command
``` npm install ```
And then
``` npm start ```
4. Your project will be running on localhost:3000
